Professional Algorithms

The same algorithms used by professional image editors, running entirely in your browser.

Best Quality

Lanczos

Sinc-windowed kernel sampling 36 pixels per output pixel. The gold standard for downscaling photography. Produces the sharpest results with minimal artifacts.

36 samples/px Sinc kernel

Bicubic

Cubic convolution interpolation using a 4x4 pixel neighborhood. Great balance of quality and performance. The default in most image editors.

16 samples/px Cubic kernel

Bilinear

Linear interpolation from 4 surrounding pixels. Fast processing with good quality for moderate size changes. Ideal for real-time previews.

4 samples/px Linear kernel
Fastest

Nearest Neighbor

Direct pixel mapping with zero interpolation. Preserves hard edges perfectly. The only choice for pixel art scaling.

1 sample/px No interpolation
